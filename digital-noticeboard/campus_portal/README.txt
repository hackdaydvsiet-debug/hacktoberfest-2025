
Campus Portal (Flask)
--------------------
Run steps:
1. Create a virtualenv: python -m venv venv
2. Activate it: source venv/bin/activate   (Windows: venv\Scripts\activate)
3. Install deps: pip install -r requirements.txt
4. Run: python app.py
5. Open http://127.0.0.1:5000/
